The project is designed to manage public charging stations in buildings.
Users can book a spot and specify the time they plan to vacate it. 
This system helps other users see when a spot will be available, 
reducing queues and conflicts caused by misunderstandings or 
irresponsible behavior.  The project is built using Python and Flask,
with a PostgreSQL database, and does not use an ORM. 
Currently, the project is deployed on Heroku and is fully functional.
Debugging and new feature development are ongoing.  


You can explore the project by downloading it and running 
_`pip install -r requirements.txt`_ to install the dependencies.

The program is lightweight with minimal dependencies, 
so you should not encounter any issues.
