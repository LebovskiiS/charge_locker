# ChargerLocker
## Description
ChargerLocker is a project designed to manage public charging stations in buildings. Users can reserve a space and specify when they plan to vacate it. This system aids other users by letting them know when a space will be available, which subsequently reduces queues and conflicts caused by miscommunication or irresponsible behavior. The project is currently deployed on Heroku and is fully functional. Debugging and the development of new features is an ongoing process.
## Technologies
- Flask
- PostgreSQL
- Heroku

## Installation
Follow the steps below to install and run the project:
1. Clone the repository:
```bash
git clone https://github.com/LebovskiiS/charge_locker.git
```

2. Install the required packages:
```bash
    pip install -r requirements.txt
```

3. Run the project:
    `python main` or run it from file main.py
